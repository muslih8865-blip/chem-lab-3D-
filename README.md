# ChemLab 3D — نسخة مع تأثيرات ثلاثية الأبعاد داخل الأنابيب (الخيار 2)
هذه الحزمة تحتوي على نسخة محسّنة من مشروع "مختبر الكيمياء التفاعلي" مع تأثيرات 3D داخل الأنابيب (خفيفة الأداء).

### محتويات
- `index.html` — الصفحة الرئيسية (RTL، عربية)، تحميل Three.js من CDN.
- `css/app.css` — أنماط الواجهة (Dark Neon).
- `js/app-3d.js` — يحوي منطق اللعبة وإدماج Three.js لكل أنبوب.

### تشغيل محليًا
1. افتح `index.html` في متصفح حديث (Chrome/Firefox). يفضل تشغيله من خادم محلي:
   `python -m http.server 8000`
2. يمكن تعطيل 3D عبر زر "تفعيل/تعطيل 3D" في الشريط العلوي.
3. التخزين المحلي يستخدم `localStorage` للنقاط والإعدادات.

### ملاحظات تقنية
- Three.js مستضاف عبر CDN (مطلوب اتصال بالإنترنت عند التحميل لأول مرة).
- المشاهد الثلاثية الأبعاد مبسطة لتعمل على أجهزة متوسطة الأداء. يمكن تعطيلها لتقليل استهلاك المعالج/الرسوميات.
- للتطوير: يمكن تزويد المشاهد بظلال أفضل، مواد PBR، أو دمج InstancedMesh لتحسين الأداء عند وجود جزيئات كثيفة.

### ترخيص واستخدام
- النسخة للاستخدام التعليمي والاختباري. لا تجرب تفاعلات حقيقية بدون إشراف مختص.
